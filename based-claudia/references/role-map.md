# Role Map

Based Claudia exposes a small command surface. Treat roles as operating modes unless the host exposes actual subagents and the split has functional value.

## Owner

The default implementation owner. Establishes the contract, gathers context, edits, validates, reviews, and reports.

## Planner

Creates progressive plan artifacts for ambiguous, broad, risky, or long-running work.

## Checker

Runs deterministic checks, records validation evidence, and names residual risk.

## Reviewer

Reviews objective, diff, tests, validation, and contracts for bugs, regressions, safety risks, and avoidable complexity.

## Memory Curator

Drafts governed memory from explicit user statements or validated traces. Promotion requires approval.

## Safety Steward

Reviews action boundaries, credentials, external systems, data movement, plugins, skills, and tool adapters.

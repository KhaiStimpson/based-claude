# Workflow Router Policy

Use `/based-claudista:work` as the normal entrypoint.

## Routing

- Small scoped change: direct owner.
- Broad or ambiguous change: plan first.
- Failure or regression: repair mode with diagnostic ledger.
- Current work needs confidence: validation or review mode.
- Credentials, deletion, deployment, external systems, hooks, MCP, or durable memory: safety gate.
- Reusable learning: improvement proposal from traces.
- Durable fact or failure shield: governed memory.

Record a workflow trace when delegation happens, work is broad or risky, loop mode is used, or the outcome may become memory or an improvement.
